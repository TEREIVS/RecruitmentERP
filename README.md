# 🎯 Recruitment ERP

### CV Management & Talent Acquisition Platform

**KM Consult Recruitment ERP** е интелигентна уеб-базирана платформа за управление на база от CV-та с AI разпознаване, многокритериално търсене, кандидатствания към компании, и GDPR съответствие.

## 📋 Описание

Recruitment ERP е цялостна платформа за HR екипи и рекрутъри, която замества хаотичното търсене в папки със CV-та с интелигентна централизирана база. Поддържа автоматично извличане на данни от PDF/DOCX файлове, силно филтриране, проследяване на кандидатствания към фирми, и AI скоринг алгоритъм за намиране на най-подходящите кандидати.

Демото в това репо е напълно функционално — съдържа 9 реални примерни CV-та, login екран със 2FA, админ панел с управление на потребители, имейл модул, бизнес визитки, календар с напомняния, и статистики.

> **🔑 Демо достъп:** Първоначалните credentials се предоставят от администратора (за сигурност не са показани публично). При първото влизане задължително се преминава през 2FA setup с Google Authenticator.

## ✨ Основни функционалности

### 🔍 Търсене и филтриране
- **15+ филтъра**: локация, езици, индустрия, йерархия, компания, сертификати, умения, образование, стаж, шофьорска книжка, релокация, и др.
- **Многокритериален синтаксис**: `София+English+ISO` намира профили съдържащи и трите критерия
- **Свободно търсене** в пълнотекстовия индекс
- **8 опции за сортиране** включително AI увереност и брой сертификати
- **Custom цветни етикети** (Обещаващ, VIP, За връщане, и др.)

### 👥 Управление на профили
- **Множество бележки с дати и автор** — пълна история
- **Кандидатствания към компании** с timeline на статуси (Непроучен → Проучен → Свързан → Интервюиран → Одобрен/Отхвърлен/Нает)
- **История на промените** — кой какво е променил по даден профил
- **Бързо четене на бележки** в popup без отваряне на пълен профил
- **Drag & drop импорт** на множество CV-та едновременно (PDF, DOC, DOCX, RTF, ODT, TXT)

### 🎯 AI и анализ
- **AI Auto Matching** — скоринг алгоритъм с настройваеми тегла за всеки критерий
- **Топ 3 медали** (злато/сребро/бронз) за най-съответстващите кандидати
- **Statistics Dashboard** с 6 SVG графики (Pie, Bar, Line) — разпределение по сфери, локация, езици, CV-та по време
- **Сравнителен механизъм** за 2-4 кандидата side-by-side с маркиране на общи и уникални характеристики

### 📧 Комуникация
- **SMTP конфигурация** с 6 готови presets (Gmail, Outlook, Yahoo, ABV.bg, Mail.bg, Custom)
- **Имейл шаблони** — Покана за интервю, Follow-up, GDPR съгласие, Учтив отказ, Обновяване на CV
- **Динамични променливи** (`{{first_name}}`, `{{position}}`, и др.) за персонализация
- **Mass email** с GDPR защита (изисква одобрение от Super Admin при 10+ получатели)
- **Бизнес визитки** за всеки потребител с снимка, лого, vCard импорт/експорт

### 🔔 Календар и напомняния
- **Todoist-style напомняния** с типове (📞 Обаждане, 🤝 Среща, ↩️ Follow-up, ⏰ Срок)
- **Bell badge** с брой просрочени/днешни в хедъра
- **Toast notification** при login с предупреждения за просрочени
- **Свързване с конкретен кандидат** + бутон за бързо добавяне от профила

### 🔒 Сигурност
- **3-степенна ролева система** (Super Admin / HR Manager / Viewer)
- **Двуфакторна автентикация** с реален TOTP (RFC 6238, HMAC-SHA1)
- **Локален QR генератор** — работи офлайн, без external API
- **10 резервни кода** при загубен телефон (eднократни)
- **Опционален 2FA bypass** за 2 часа на доверено устройство
- **Session timeout** 10 минути с auto-reset при активност
- **Audit Log** на всяко действие (login, view, edit, delete, import, export, email)
- **Валидатор за сложност на парола** (A-Z, 0-9, специален знак, мин. 8 символа)

### ⚙️ Администрация
- **7 раздела в админ панел**: Потребители, Роли, Фирми, Audit Log, Backup, Имейл сървър, Настройки
- **CRUD на потребители** с verify-after-save персистентност
- **Companies database** с асоциация към CV-та
- **Backup стратегия** — HR Manager може да стартира backup, Restore само за Super Admin
- **GDPR панел** с период на съхранение, известия, политики

## 🛠️ Технологичен стек

| Слой | Технология |
|------|-----------|
| **Frontend** | Pure HTML5 + CSS3 + Vanilla JavaScript |
| **2FA TOTP** | RFC 6238 (HMAC-SHA1 чрез Web Crypto API) |
| **QR Code** | Собствен JS encoder (ISO/IEC 18004, Reed-Solomon) |
| **Графики** | Inline SVG (без външни библиотеки) |
| **Storage** | Browser localStorage (для демо), готов за PostgreSQL миграция |
| **Зависимости** | **0 external** — всичко работи офлайн |

Просто отворете публичния URL:

🌐 **https://tereivs.github.io/RecruitmentERP/**

### Опция 2: Локално

```bash
# Клонирайте repo-то
git clone https://github.com/TEREIVS/RecruitmentERP.git

# Отворете index.html в браузъра (двоен клик)
```

```bash
# С Python
python -m http.server 8000

# С Node.js
npx serve

# Отворете http://localhost:8000
```

## 🎯 Roadmap (за продукция)

Демото демонстрира концепцията. За реална production среда:

- [ ] **Backend** — Python FastAPI + PostgreSQL + Redis
- [ ] **AI/NLP** — spaCy NER + локален LLM (Llama/Mistral) за извличане от CV-та

- [ ] **GDPR панел** — реални процедури за изтриване и експорт
- [ ] **Audit Log в DB** — пълно проследяване на ниво потребител
